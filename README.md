## 使用说明

- 启用脚本后打开雀魂即可自动加载

- 此脚本同时提供最近大铳和最近炸庄两个功能

- 数据会在用户页原最近大和区域循环显示

- 目前用户较少，因为放铳数据都是安装此脚本的用户上传的，所以基本上匹配到的人都没有放铳数据（
